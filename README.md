# Gestão de vencimentos

Aplicação web para gerenciar vencimentos de produtos em um comércio, com autenticação de usuários

## 🚀 Tecnologias

Node.js
Express
Sequelize
MySql
Handlebars
BCryptjs

### Pré-requisitos
- Node.js instalado
- MySQL rodando

## Vídeo de demonstração
https://www.youtube.com/watch?v=hyj4mV2WXlQ

### Passos

# Clone o repositório
git clone https://github.com/seu-usuario/seu-repo.git

# Entre na pasta
cd seu-repo

# Instale as dependências
npm install

# Configuração banco de dados
crie um banco de dados vazio com o nome 'vencimentos'

# Rode o projeto
npm start
